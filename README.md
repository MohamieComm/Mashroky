# Mashrouk Travel & Tourism

مشروع تطبيق (مشروك) للسفر والسياحة باللغة العربية، موجه للسعودية والشرق الأوسط.

## الهيكل
- `app/` الواجهة (Vite + React + Tailwind) مع صفحات وخدمات المنصة.
- `app/src/main/` مشروع Android (Compose) كبنية تطبيق هجين.
- `public/` ملفات التصميم المرجعية (PDFs) والمواد المرفقة.

## التشغيل المحلي (واجهة الويب)
```bash
cd app
npm install
npm run dev
```

## ملاحظات
- الفيديو الدعائي للواجهة الرئيسية يُمرّر عبر `VITE_PROMO_VIDEO_URL` كمتغير بيئة عند الحاجة.
- ملف الشعار المستخدم في الواجهة: `app/public/logo.png`.
